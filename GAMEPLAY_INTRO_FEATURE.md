### 功能1：角色玩法简介

#### HTML结构更新

#### UI渲染逻辑（ui.js）

```javascript
el.innerHTML = `
    ${avatarHtml}
    <div class="hero-name">${hero.name}</div>
    <div class="hero-skill">${hero.skill}</div>
    <div class="hero-skill-brief">${hero.skillBrief || hero.skillDesc}</div>
    <div class="hero-hp">${hero.hp}HP</div>
`;
```

**智能降级**：如果没有 `skillBrief`，自动使用 `skillDesc`

#### CSS样式新增

```css
/* 桌面端 */
.hero-skill-brief {
    font-size: 7px;
    color: #95a5a6;
    margin-top: 2px;
    line-height: 1.2;
    max-width: 90%;
    word-break: break-word;
}

/* 移动设备 */
@media (max-width: 480px) {
    .hero-skill-brief { font-size: 6px; }
}
```

#### 视觉改进

- **卡牌高度**：固定65px → 自适应 (min-height: 80px)
- **文字颜色**：简介使用较浅灰色 (#95a5a6)，区分于技能名称
- **字体大小**：7px（桌面）/ 6px（移动），保持可读性
- **行间距**：1.2倍行高，防止文字过密

## 💡 用户体验改进

### 对新玩家的帮助

1. **快速了解角色定位**
   - 不需要打开帮助文档
   - 直观了解每个角色的目标
   - 帮助选择合适的身份

2. **快速了解英雄特点**
   - 一句话总结英雄的玩法风格
   - 帮助选择喜欢的玩法
   - 快速上手游戏

### 信息层级优化

**角色卡牌**：
- Level 1（主要）：角色名称
- Level 2（说明）：玩法简介

**武将卡牌**：
- Level 1（主要）：英雄名称 + 头像
- Level 2（说明）：技能名称
- Level 3（辅助）：技能简介 + 血量

## 🔄 向后兼容性

### skillBrief 字段

- **可选字段**：不是必须的
- **智能降级**：如果缺失，自动使用 `skillDesc`
- **新增英雄**：添加新英雄时可以不填 `skillBrief`

```javascript
// 示例：如果添加新英雄忘记填 skillBrief
const newGeneral = {
    name: '新英雄',
    skill: '新技能',
    skillDesc: '技能效果描述'
    // 没有 skillBrief，会自动使用 skillDesc
};
```
