## 🚀 技术细节

### 设计原则
1. **一致性缩放**: 所有屏幕尺寸的卡片都按相同比例增大
2. **响应式优化**: 不同屏幕尺寸有针对性的调整
3. **可读性优先**: 确保所有文字都清晰可见
4. **视觉平衡**: 头像、文字、卡片高度等元素保持协调

### 具体改动

#### 敌方卡片
```css
/* 基础 */
.enemy-card { width: 32%; } /* 28% → 32% */
.avatar-frame { font-size: 56px; } /* 48px → 56px */

/* 所有媒体查询已统一为 32% */
```

#### 英雄卡片 (基础)
```css
.hero-option { height: 220px; } /* 180px → 220px */
.hero-avatar { font-size: 56px; } /* 40px → 56px */
.hero-avatar-img { width: 80px; height: 80px; } /* 60px → 80px */
.hero-name { font-size: 14px; } /* 11px → 14px */
.hero-skill { font-size: 11px; } /* 8px → 11px */
```

#### 英雄卡片 (600px 断点)
```css
.hero-option { height: 180px; } /* 140px → 180px */
.hero-avatar { font-size: 40px; } /* 24px → 40px */
.hero-name { font-size: 13px; } /* 12px → 13px */

```
