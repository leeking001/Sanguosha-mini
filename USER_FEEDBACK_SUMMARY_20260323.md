## 💻 技术亮点

### 1. 智能降级机制
```javascript
// 如果缺少 skillBrief，自动使用 skillDesc
<div class="hero-skill-brief">${hero.skillBrief || hero.skillDesc}</div>
```

### 2. 响应式设计
```css
/* 桌面：7px */
.hero-skill-brief { font-size: 7px; }

/* 移动：6px */
@media (max-width: 480px) {
    .hero-skill-brief { font-size: 6px; }
}
```

### 3. 自适应卡牌高度
```css
/* 之前：固定高度 */
.hero-option { height: 65px; }

/* 之后：自适应 */
.hero-option {
    height: auto;
    min-height: 80px;
}
```

## 📞 技术支持

### 常见问题

**Q: 为什么我的英雄没有显示技能简介？**
A: 检查 `generals.js` 中是否有 `skillBrief` 字段。如果没有，会自动使用 `skillDesc`。

**Q: 移动设备上简介文字太小了？**
A: 这是设计的，为了在小屏幕上节省空间。如需调整，编辑 `style.css` 中的移动媒体查询。

**Q: 我想添加新的角色/英雄？**
A:
1. 添加到对应的数据结构
2. 为英雄添加 `skillBrief` 字段
3. 刷新页面查看效果
