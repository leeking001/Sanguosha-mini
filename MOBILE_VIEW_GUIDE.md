# 迷你杀 - PC 手机视图模式使用指南

## 🎮 快速开始

### 如何启用手机视图？

1. **打开游戏** - 在 PC 浏览器中打开迷你杀游戏
2. **点击手机按钮** - 在顶部右上角找到 **📱** 按钮（在语言按钮左侧）
3. **完成！** - 游戏界面自动调整为手机视图

### 如何关闭手机视图？

- 再次点击 **📱** 按钮即可恢复桌面视图

## 📱 视图对比

### 桌面视图
- 全屏显示
- 利用整个浏览器窗口
- 适合宽屏显示器

### 手机视图
- 固定宽度 480px（标准手机宽度）
- 居中显示
- 金色边框标识
- 所有元素按手机端优化
- 更好的可读性和易用性

## 🎯 适配内容

手机视图模式完整适配了游戏的所有界面：

✅ **欢迎页面** - 游戏标题、介绍、开始按钮
✅ **身份选择** - 主公、忠臣、反贼、内奸
✅ **英雄选择** - 2 列网格布局
✅ **游戏战场** - 敌方区域、日志、玩家区域、卡牌
✅ **结算画面** - 最终结果统计
✅ **帮助弹窗** - 游戏说明书

## 💾 自动保存

- 你的视图选择会自动保存
- 下次访问游戏时会自动恢复你上次的选择
- 不需要每次都重新设置

## 🎨 视觉效果

### 手机视图特征
- **金色边框** - 清晰标识当前处于手机视图
- **居中显示** - 在屏幕中央呈现
- **光晕效果** - 金色阴影增强视觉效果
- **紧凑布局** - 所有元素优化以适应 480px 宽度

## 📊 尺寸调整

启用手机视图时的自动调整：

| 元素 | 调整 |
|------|------|
| 卡牌 | 70x98px |
| 顶部栏 | 高度 40px |
| 敌方区域 | 最大高度 25vh |
| 英雄选择 | 2 列（而非 3 列） |
| 文字大小 | 整体缩小 10-20% |
| 按钮 | 紧凑间距 |

## ❓ 常见问题

**Q: 手机视图会影响游戏功能吗？**
A: 不会。这只是视觉调整，所有游戏功能完全相同。

**Q: 如何在移动设备上使用？**
A: 移动设备已经是最优化的布局，手机视图按钮主要为 PC 用户设计。

**Q: 可以自定义视图宽度吗？**
A: 当前固定为 480px。如需自定义，可以联系开发者。

**Q: 视图设置会在浏览器关闭后保存吗？**
A: 是的，使用 localStorage 保存，即使关闭浏览器也会保留。

**Q: 在手机视图中可以全屏吗？**
A: 可以。按 F11（或浏览器全屏快捷键）进入全屏模式。

## 🔧 技术信息

- **实现方式** - CSS 响应式设计 + JavaScript 切换
- **存储方式** - 浏览器 localStorage
- **兼容性** - 所有现代浏览器
- **性能影响** - 无（仅 CSS 类切换）

## 💡 建议

- 在 PC 上使用手机视图可以获得最佳的游戏体验
- 特别推荐在笔记本电脑上使用，可以减少眼睛疲劳
- 可以根据个人偏好在桌面和手机视图之间切换
