# 🀄 三国杀 H5 单机版

**HTML5 / JavaScript 单文件版 · 无需服务器 · 可离线运行**

一个极简但功能完整的 **三国杀单机版**：无需安装环境、无需联网、无需资源文件，只用一个 HTML 文件，即可体验武将技能、身份模式、音效、卡牌逻辑和智能 AI。

---

## 🌟 特性亮点

### 🎨 武侠风 UI

* 水墨背景、羊皮纸纹理
* 古铜色 UI 元素
* 中文宋体 / 楷体风格
* 卡牌、头像全面重制

### 🎮 完整的三国杀玩法

* 基础卡牌：杀 / 闪 / 桃 / 万箭齐发 / 南蛮入侵 / 无中生有
* 武将系统（吕布、曹操、郭嘉、张飞、赵云、华佗等）
* AI 行为：智能判断攻击、回血、闪避、控场
* 4 人局身份系统：主公 / 忠臣 / 反贼 / 内奸

### 📱 手机竖屏深度优化

* 顶部 AI 平铺
* 中间日志可滑动
* 底部卡牌横向滑动（炉石风）
* 大按钮方便点击

### 🔊 内置原生音效（无需 mp3）

* 出杀、受伤、回血、AOE
* 胜利 / 失败 8-bit 旋律
* Web Audio API 原生合成

---

## 🕹️ 快速开始

### ▶️ **方式一：直接双击运行（推荐）**

下载 `index.html` → 双击打开 → 开始游戏
（无需任何依赖、无需服务器）

### ▶️ **方式二：部署到 GitHub Pages**

1. 新建仓库
2. 上传整个项目（或一个 index.html）
3. 打开 **Settings → Pages**
4. Source 选择 `main / root`
5. 等待 10 秒即可访问在线版本

## 🔔 音效说明

点击右上角 **🔊 按钮** 开启/关闭音效。

| 行为      | 音效       |
| ------- | -------- |
| 出牌      | 纸张摩擦声    |
| 杀       | 刀剑声      |
| 受伤      | 低沉撞击声    |
| 南蛮 / 万箭 | 呼啸声      |
| 胜利/失败   | 8-bit 旋律 |

全部由 **Web Audio API 原生合成**，无需下载 mp3。
