0.05版

核心升级内容：
武侠风 UI 重制：
背景改为水墨暗色调，字体改为宋体/楷体。
卡牌和界面采用羊皮纸纹理色调。
按钮和边框采用古铜色/金色风格。
开局选将：
游戏开始前，会出现 4 张武将卡，你可以任选其一作为主公，剩余的分配给 AI。
音效系统 (Web Audio API)：
为了保持单文件且不依赖外部资源，我用 JavaScript 原生合成音效（无需下载 mp3）。
包含：出牌声、杀戮声、受伤声、回血声、游戏胜利/失败音效。
日志优化：
战斗日志现在直接显示武将名字（如“曹操对张飞出杀”），而不是“AI-1”。

🔊 音效说明
开启音效：点击右上角的 🔊 按钮。
声音类型：
出牌：轻微的纸张摩擦声。
杀：锐利的刀剑声。
受伤：低沉的打击声。
AOE (万箭/南蛮)：连续的呼啸声。
胜利/失败：复古的 8-bit 风格旋律。

⚔️ 新增武将
吕布：【无双】出杀时，有 50% 概率无视对方的闪（模拟强力攻击）。
华佗：【急救】（简化版）血量较低时，AI 会更倾向于留桃。
现在，你可以体验一个更有沉浸感的“掌上三国杀”了！

0.03版

这是一个针对手机竖屏体验深度优化的版本。

主要优化点：
布局重构：放弃了“左中右”布局，改为上中下结构。
顶部：3个 AI 一字排开，节省空间。
中间：战斗日志，限制高度，可滑动，不会遮挡操作。
底部：玩家区域，手牌改为横向滑动（类似炉石传说手游），防止手牌过多挤压按钮。
操作优化：按钮固定在最底部，更大、更易点击。
视觉调整：字体调大，卡牌在手机上显示更紧凑。

手机版操作指南
选择目标：
点击手牌【杀】。
上方 3 个 AI 的卡片会闪烁红色光圈。
直接点击你想攻击的 AI 卡片即可出牌。
手牌管理：
如果手牌很多，在屏幕下方的卡牌区域左右滑动即可查看所有牌。
日志查看：
中间的文字区域可以上下滑动，查看之前的出牌记录。
按钮：
出牌：用于【桃】（给自己吃）和【万箭齐发】（放全场）。
取消：取消当前选中的牌。
结束：回合结束。

0.02版

这是一个大幅升级的版本。为了实现 4人局（1主公 vs 1忠 vs 1反 vs 1内） 以及 武将技能，代码逻辑变得更加复杂。

这个版本包含了：
4人布局：你（主公）在下方，其他3个AI分布在左、上、右。
身份系统：
你是主公（亮明身份）。
AI 随机分配 忠臣、反贼、内奸（身份隐藏，只有死掉才会亮出）。
武将与技能：
曹操（你）：【奸雄】受伤后立即摸一张牌。
张飞：【咆哮】出杀没有次数限制。
郭嘉：【遗计】受伤后立即摸两张牌。
赵云：【龙胆】防御力更强（简化版：手牌上限+1，且AI出闪概率更高）。
AI 智能：
反贼优先打主公。
忠臣优先打攻击过主公的人。
内奸随机控场。

玩法说明
布局：
下方：是你（主公，曹操）。
左、上、右：是三个AI，身份分别为忠臣、反贼、内奸（随机分配，隐藏）。
操作：
出杀：点击手牌中的【杀】 -> 牌会浮起 -> 点击场上任意一个红色边框闪烁的AI头像 -> 确认攻击。
吃桃：点击手牌中的【桃】 -> 点击“出牌”按钮（给自己回血）。
万箭齐发：点击【万箭】 -> 点击“出牌”按钮（全场除你之外的人需出闪）。
结束回合：没有牌出时，点击“结束回合”。
技能：
如果你（曹操）受到伤害，会自动触发【奸雄】，摸一张牌。
如果AI是郭嘉，打他一下他会摸两张牌（很难缠）。
如果AI是张飞，他可能会在一回合内对你出好几张杀。
胜利条件：
胜利：反贼和内奸全部阵亡。
失败：你（主公）阵亡。

0.01版

这是一个完整的、单文件的 HTML5 版三国杀（1v1 单机对战）。
它包含了游戏逻辑（JavaScript）、界面样式（CSS）和结构（HTML）。你只需要下载这个文件，不需要安装任何环境，直接双击打开就能玩，也可以直接上传到服务器。

核心功能：
卡牌系统：杀（攻击）、闪（自动抵消攻击）、桃（回血）、万箭齐发（强力攻击）。
AI 逻辑：电脑会自动出牌、自动吃桃、自动出闪。
胜负判定：一方血量归零游戏结束。
战斗日志：实时显示由于什么操作导致了扣血或抵消。
