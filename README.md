# 🀄 三国杀 H5 单机版

**HTML5 / JavaScript 模块化 · 无需服务器 · 可离线运行**

一个极简但功能完整的 **三国杀单机版**：无需安装环境、无需联网、无需资源文件，纯前端实现，即可体验完整武将技能、身份模式、音效卡牌逻辑和智能 AI。

## 🎴 卡牌说明

### 基础牌

| 卡牌 | 效果 |
|------|------|
| ⚔️ **杀** | 攻击范围内的一名角色，需用【闪】抵消 |
| 💨 **闪** | 抵消【杀】或【万箭齐发】的效果 |
| 🍑 **桃** | 回合内回复1点体力；濒死时可救自己或他人 |
| 🍶 **酒** | 回合内使用，本回合下一张杀伤害+1；或濒死时自救 |

### 锦囊牌

| 卡牌 | 效果 |
|------|------|
| 🏹 **万箭齐发** | 所有其他角色需打出一张【闪】，否则受1点伤害 |
| 🐘 **南蛮入侵** | 所有其他角色需打出一张【杀】，否则受1点伤害 |
| 🎁 **无中生有** | 摸2张牌 |
| 🌾 **五谷丰登** | 全场每人摸1张牌 |
| 🔗 **顺手牵羊** | 获得距离1以内的一名角色一张牌 |
| 🪓 **过河拆桥** | 弃置任意一名角色一张牌 |
| ⚔️ **决斗** | 与目标轮流出【杀】，不出者受1点伤害 |
| 🔥 **火攻** | 弃置一张手牌，目标需弃置一张牌，否则受1点伤害 |
| ⛓️ **铁索连环** | 选择两名角色。处于连环状态的角色受伤害时，其他连环角色同受伤害 |
| 🤐 **乐不思蜀** | 延时锦囊。判定阶段若判定不为红桃，则跳过出牌阶段 |
| 🍚 **兵粮寸断** | 延时锦囊。判定阶段若判定不为梅花，则跳过摸牌阶段 |
| ⚡ **闪电** | 延时锦囊。回合开始判定，若为黑桃-2到黑桃-9则受到3点伤害，否则移动到下家 |
| 🛡️ **无懈可击** | 抵消一张锦囊牌对一名角色的效果 |
| 🔪 **借刀杀人** | 选择一名有武器的角色，令该角色对另一名角色使用【杀】，否则你获得其武器 |
| 🌸 **桃园结义** | 所有角色回复1点体力 |

## 🎮 游戏流程

1. **第一步：选择你的身份** - 主公/忠臣/反贼/内奸四选一
2. **第二步：选择你的英雄** - 从随机生成的英雄中选择一个
3. **开始游戏** - 轮流进行回合
   - **摸牌阶段**：摸两张牌（技能可能改变数量）
   - **出牌阶段**：出牌、使用技能
   - **弃牌阶段**：弃牌到手牌上限
   - **结束回合**

### 操作说明

- **选牌**：点击手牌选中，再次点击取消
- **选目标**：选中卡牌后点击目标角色
- **出牌**：选好牌和目标后点击「出牌」按钮（无目标卡牌如桃、酒、无中生有点击「确认」）
- **取消**：点击「取消」取消选择
- **结束**：点击「结束」结束回合
- **技能触发**：特殊技能（华佗、孙权）点击自己头像触发

## 🔔 音效说明

点击右上角 **🎵** 按钮开启/关闭背景音乐，**🔊** 按钮开启/关闭音效。

| 行为      | 音效       |
| ------- | -------- |
| 出牌      | 纸张摩擦声    |
| 杀       | 刀剑声      |
| 受伤      | 低沉撞击声    |
| 南蛮 / 万箭 | 呼啸声      |
| 胜利/失败   | 8-bit 旋律 |

全部由 **Web Audio API 原生合成**，无需下载 mp3。
