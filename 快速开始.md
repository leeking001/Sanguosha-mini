# ✅ 三国杀mini - 项目已修复！

## 🎯 问题原因
项目使用 **ES6 模块系统**，必须通过 HTTP 服务器运行，不能直接双击打开。

## 🚀 立即开始游戏

```bash
./manage.sh start
```

游戏会自动在浏览器打开：http://localhost:8000

## 📋 常用命令

| 命令 | 功能 |
|------|------|
| `./manage.sh start` | 启动游戏 |
| `./manage.sh stop` | 停止服务器 |
| `./manage.sh restart` | 重启服务器 |
| `./manage.sh status` | 查看状态 |
| `./manage.sh open` | 打开游戏页面 |
