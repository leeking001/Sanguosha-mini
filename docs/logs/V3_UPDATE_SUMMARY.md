## 🔧 技术亮点

### i18n 设计

```
┌─────────────────────────────┐
│    i18n.js (翻译系统)       │
├─────────────────────────────┤
│  translations:              │
│  ├─ zh: {...}               │
│  └─ en: {...}               │
├─────────────────────────────┤
│  方法:                       │
│  ├─ t() - 翻译获取          │
│  ├─ setLanguage() - 设置    │
│  ├─ toggleLanguage() - 切换 │
│  └─ getLanguage() - 获取    │
└─────────────────────────────┘
         ↓
┌─────────────────────────────┐
│  index.html (UI 更新)       │
├─────────────────────────────┤
│  updateUILanguage()         │
│  - 更新所有文本             │
│  - 同步按钮标签             │
│  - 刷新界面                 │
└─────────────────────────────┘
         ↓
┌─────────────────────────────┐
│  localStorage (持久化)      │
├─────────────────────────────┤
│  sanguosha_lang: 'zh'/'en' │
└─────────────────────────────┘
```

### 玩家视角

```
打开游戏
    ↓
点击 🌐 按钮
    ↓
选择语言 (zh/en)
    ↓
界面立即更新
    ↓
语言偏好自动保存
    ↓
下次打开保持选择
```

### 开发者视角

```
需要新文本?
    ↓
在 i18n.js 中添加翻译键值对
    ↓
同时添加中英文翻译
    ↓
在代码中调用 I18N.t('key')
    ↓
完成！自动支持多语言
```
