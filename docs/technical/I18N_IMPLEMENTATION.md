## 🛠️ 技术实现细节

### i18n 模块架构

```javascript
I18N.t(key, ...args)           // 获取翻译文本，支持参数替换
I18N.setLanguage(lang)          // 设置语言（'zh' 或 'en'）
I18N.getLanguage()              // 获取当前语言
I18N.toggleLanguage()           // 切换语言
```

#### 示例用法

```javascript
// 基础翻译
I18N.t('title')                    // 返回：迷你杀·英雄对决 / Mini Sha - Legendary Duel

// 带参数的翻译
I18N.t('draw_phase', '大耳备', 2)  // 返回：大耳备进入摸牌阶段，摸了2张牌

// 设置语言
I18N.setLanguage('en')            // 切换到英文
I18N.toggleLanguage()             // 在中英文之间切换
```

#### 语言文件结构

每个语言都有对应的翻译对象：

```javascript
translations: {
    zh: {
        'title': '迷你杀·英雄对决',
        'card_sha': '杀',
        'hero_daerbei': '大耳备',
        // ... 更多翻译
    },
    en: {
        'title': 'Mini Sha - Legendary Duel',
        'card_sha': 'Sha',
        'hero_daerbei': 'Big Ear',
        // ... 更多翻译
    }
}
```

### UI 更新机制

当用户点击语言切换按钮时：

1. 点击 🌐 按钮
2. `initLanguageToggle()` 触发 `I18N.toggleLanguage()`
3. I18N 分发 `languageChanged` 事件
4. `updateUILanguage()` 监听事件并更新所有 UI 文本
5. 语言选择自动保存到 localStorage

### 集成点

在 `index.html` 中：

```javascript
// 导入 i18n 模块
import { I18N } from './i18n.js';

// 初始化语言切换
initLanguageToggle();

// 监听语言变更
window.addEventListener('languageChanged', (e) => {
    updateUILanguage();
});
```

## 🔄 扩展语言的步骤

如果要添加其他语言（如日文、韩文等），只需：

1. 在 `i18n.js` 的 `translations` 对象中添加新语言代码
2. 为每个翻译键添加该语言的翻译文本
3. 语言会自动可用

示例（添加日文）：

```javascript
translations: {
    zh: { /* 中文翻译 */ },
    en: { /* 英文翻译 */ },
    ja: {  // 新增日文
        'title': 'ミニ殺 - 伝説の決闘',
        'card_sha': '殺',
        // ... 更多翻译
    }
}
```
