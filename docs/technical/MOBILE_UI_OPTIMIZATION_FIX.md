## 用户反馈问题

### 1. ❌ 手机上选武将界面，还是无法一屏展现
**问题描述:** 英雄选择界面在手机上需要滚动才能看到全部

**解决方案:**
- 减少 `.hero-options` 的 `max-height` 从 **50vh** → **45vh**
- 减少 `.hero-option` 的高度从 **110px** → **95px**
- 减少间距 `gap` 从 **8px** → **6px**
- 减少 padding 从 **5px** → **4px**

**影响范围:**
- 触屏设备: `@media (hover: none) and (pointer: coarse)`
- 手机屏幕: `@media (max-width: 768px)`
- PC 手机视图: `@media (min-width: 600px)`

✅ **预期结果:** 4 个英雄可在手机一屏显示，无需滚动

---

### 2. ❌ 手机上战斗界面，玩家的血格变形看不见了
**问题描述:** HP 血格显示异常，数字过多导致排列混乱

**解决方案:**

| 属性 | 之前 | 之后 | 作用 |
|------|------|------|------|
| `.hp-bar` 宽度 | 无限制 | `max-width: 180px` | 限制最大宽度 |
| `.hp-bar` 换行 | 无 | `flex-wrap: wrap` | 自动换行显示 |
| `.hp-bar` gap | `4px` | `3px` | 减少间距 |
| `.hp-point` 尺寸 | `16x16px` | `14x14px` | 整体缩小 |
| `.hp-point` 边框 | `2px` | `1.5px` | 保持比例 |
| `.hp-lost` 属性 | - | `flex-shrink: 0` | 防止挤压 |

**手机端调整:**
- 触屏: `.hp-point` 改为 `13x13px` 和 `1.5px` 边框
- 手机: `max-width: 160px`

✅ **预期结果:** 血格正常显示，支持自动换行

---

### 3. ❌ 战斗记录太空了，右边有很大空白
**问题描述:** 日志区域右侧有大量空白，文字不够紧凑

**解决方案:**

#### 日志区域 (`#log-zone`)
| 属性 | 之前 | 之后 | 作用 |
|------|------|------|------|
| `padding` | `12px 18px` | `10px 12px` | 减少左右边距 |
| `min-height` | `100px` | `90px` | 更紧凑 |
| `font-size` | `12px` | `12px` | 保持可读性 |
| `display` | 无 | `flex` | 自动布局 |
| `flex-direction` | - | `column` | 垂直排列 |

#### 日志行 (`.log-line`)
| 属性 | 之前 | 之后 | 作用 |
|------|------|------|------|
| `margin-bottom` | `6px` | `4px` | 减少行距 |
| `line-height` | `1.5` | `1.4` | 紧凑行高 |
| `word-break` | 无 | `break-word` | 自动换行 |

#### 手机端 (`@media (hover: none)`)
- `padding`: `12px 18px` → `8px 10px`
- `font-size`: `12px` → `11px`
- `min-height`: `100px` → `85px`

✅ **预期结果:** 日志紧凑显示，右侧空白消除
