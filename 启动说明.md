# 三国杀mini - 启动说明 🎮

## 问题诊断 ✅

项目无法启动的原因是：**使用了ES6模块（import/export），必须通过HTTP服务器运行，不能直接双击打开HTML文件。**

## 快速启动 🚀

### 方法一：使用启动脚本（推荐）

```bash
./start-server.sh
```

然后在浏览器中打开: http://localhost:8000

### 方法二：手动启动

```bash
# 进入项目目录
cd ~/Claude\ Code/Sanguosha-mini

# 启动HTTP服务器
python3 -m http.server 8000

# 然后在浏览器中打开: http://localhost:8000
```

## 停止服务器 ⛔

按 `Ctrl + C` 停止服务器

或者使用命令：
```bash
# 查找并杀死占用8000端口的进程
lsof -ti:8000 | xargs kill -9
```

## 当前状态 ✨

- ✅ HTTP服务器已启动
- ✅ 游戏已在浏览器中打开
- ✅ 访问地址: http://localhost:8000
- ✅ 进程运行中

## 技术说明 📚

项目使用ES6模块系统：
- `index.html` 使用 `<script type="module">`
- 导入了多个JavaScript模块: generals.js, cards.js, game.js, ai.js, ui.js 等
- ES6模块必须通过HTTP/HTTPS协议加载，不支持file://协议
- 这是浏览器的CORS安全限制

## 故障排查 🔧

### 端口被占用
```bash
# 查看8000端口占用情况
lsof -i:8000

# 杀死占用端口的进程
lsof -ti:8000 | xargs kill -9
```

### 浏览器缓存问题
按 `Cmd + Shift + R` (Mac) 强制刷新页面

### 控制台错误
按 `F12` 或 `Cmd + Option + I` 打开开发者工具查看错误信息
