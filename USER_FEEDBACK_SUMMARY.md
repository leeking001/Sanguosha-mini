   ```javascript

   ```

   ```javascript
   // 在 handleSkillUse() 中添加
   if (result.reason === 'need_target') {
       // 显示提示信息
       // 高亮可选目标
       // 等待用户选择
   }
   ```

3. **目标选择**
   ```javascript
   // 新增函数
   handleSkillTargetSelect(targetId) {
       // 验证目标
       // 调用 useActiveSkill(0, targetId)
       // 处理结果
   }
   ```

4. **UI 更新**
   ```javascript
   // 在 renderEnemyZone() 中
   // 添加 skill-target-available 类
   // 显示蓝色脉冲高亮
   ```

5. **样式**
   ```css

       border-color: #3498db;

   ```

### 问题 2: 结算画面统计数据

#### 需求分析
用户希望看到更多维度的数据，以便：
- 评估每个角色的表现
- 找到表现出色的角色
- 分析游戏策略

#### 新增数据项

| 数据项 | 字段名 | 说明 | 颜色 |
|--------|--------|------|------|
| 伤害 | damageDealt | 造成的伤害总数 | 红色 |
| 回复 | healed | 治疗的生命总数 | 绿色 |
| 击杀 | kills | 消灭的敌方数量 | 黄色 |
| **出牌** | **cardsPlayed** | **使用的卡牌总数** | **蓝色** |
| **锦囊** | **strategiesUsed** | **使用的策略卡牌数** | **紫色** |
| **技能** | **skillsUsed** | **释放的技能总数** | **青色** |

#### 实现细节

1. **初始化** (game.js)
   ```javascript
   stats: {
       damageDealt: 0,
       healed: 0,
       kills: 0,
       cardsPlayed: 0,      // NEW
       strategiesUsed: 0,   // NEW
       skillsUsed: 0        // NEW
   }
   ```

2. **追踪出牌数** (game.js - useCard)
   ```javascript
   source.stats.cardsPlayed += 1;
   ```

3. **追踪锦囊数** (game.js - useCard 中每个策略卡)
   ```javascript
   case '万箭':
   case '南蛮':
   case '五谷':
   case '无中':
   // ... 等等
       source.stats.strategiesUsed += 1;
   ```

4. **追踪技能数** (game.js - useActiveSkill)
   ```javascript
   player.stats.skillsUsed += 1;
   ```

5. **UI 显示** (ui.js - showGameResult)
   ```javascript
   // 显示所有 6 个统计数据
   // 使用 res-stat-small 类紧凑显示
   ```

6. **样式** (style.css)
   ```css
   .res-stat-small {
       width: 11%;
       text-align: center;
       font-size: 11px;
   }
   ```

✅ 所有数据正确追踪和显示

## 💡 数据分析示例

### 识别不同类型的英雄

| 类型 | 特征 | 例子 |
|------|------|------|
| 高效能 | 高伤害 + 低出牌 | 伤害12, 出牌8 |
| 控场型 | 高锦囊 + 低伤害 | 伤害3, 锦囊5 |
| 活跃型 | 高技能 + 高出牌 | 技能3, 出牌14 |
| 治疗型 | 高回复 + 低伤害 | 回复4, 伤害3 |

### 评估英雄强度

```
总体表现 = (伤害 × 0.3) + (击杀 × 0.4) + (回复 × 0.2) + (技能 × 0.1)

示例:
诸葛亮: (12 × 0.3) + (2 × 0.4) + (1 × 0.2) + (3 × 0.1) = 6.5 分
华佗:   (3 × 0.3) + (0 × 0.4) + (4 × 0.2) + (2 × 0.1) = 1.9 分
```
